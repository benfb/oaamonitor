Outs Above Average Monitor is a website which displays the daily Outs Above Average total for MLB players. This data is viewable on Baseball Savant for each player, but this website makes it easier to view the daily changes for each player. The data is entirely from Baseball Savant. I don't think viewing defensive metrics on a daily basis is actually useful, but it's fun.
